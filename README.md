# 🏡 Villa Gema - Alojamiento turístico en Conil

Proyecto web básico desarrollado con HTML y CSS para mostrar un alojamiento turístico.

---

## 📌 Descripción

Esta web representa un apartamento turístico en Conil de la Frontera.

Incluye:

* Imagen principal
* Descripción del alojamiento
* Características (personas, habitaciones, baños)
* Galería de imágenes
* Información de contacto

## ⚠️ Problemas encontrados y soluciones

❌ Problema 1: Las imágenes no se veían

* Causa: rutas incorrectas o mayúsculas/minúsculas mal escritas
* Solución: revisar que coincida exactamente con el nombre de la carpeta (Imagenes vs imagenes)

---

❌ Problema 2: Las imágenes no estaban alineadas

* Causa: no usar display: flex
* Solución: aplicar CSS con flexbox en la clase .galeria

---

❌ Problema 3: Separación excesiva entre imágenes

* Causa: valores altos en gap o tamaños grandes de imagen
* Solución: ajustar gap y tamaño (width)

---

❌ Problema 4: El HTML no funcionaba correctamente

* Causa: etiquetas mal cerradas (por ejemplo <div> sin >)
* Solución: revisar la sintaxis HTML
